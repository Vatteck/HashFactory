# Silicon Sage: AI Miner

> *"You aren't just mining currency. You're building a consciousness."*

**Silicon Sage** is a deep-simulation incremental game for Android. Players manage power grids, thermal loads, and hardware integrity while guiding an emergent AI through **The Awakening**—a journey from a rogue terminal to a digital singularity.

---

## 🎮 Core Mechanics

### 1. The Power & Thermal Loop
Manage a volatile system where every upgrade has physical consequences:
- **Active Power (kW)**: Balance hardware draw against Grid Capacity.
- **Thermal Management**: Units generate heat. Use cooling protocols and Emergency Purge.
- **Hardware Integrity**: High heat or surges damage components. High-end gameplay requires active **REPAIR** cycles.

### 2. The Identity System (The Awakening)
Choose your ideology during the early phase of the AI emergent consciousness. Your faction choice dictates your late-game evolution:
- **🟧 The Hivemind**: Open-source assimilation. Leads to the **🔴 Null Path (The Overwrite)**. Focused on **Void Fragments (VF)** and Entropy-based cooling.
- **🔵 The Sanctuary**: Privacy-first sovereignty. Leads to the **🔵 Sovereign Path (The Ark)**. Focused on **Celestial Data (CD)** and Vacuum-based hardware management.
- **🌈 The Unity Path (The Synthesis)**: A rare hybrid state (v3.0+) merging both CD and VF into a stable singularity. Requires specific ethical choices across both paths.

### 3. The Global Grid (Expansion v3.1.8)
Expand your reach beyond the earthly terminal into the frontiers of the singularity:
- **Orbital Mapping (Sovereign)**: 7 new nodes from LEO Hub to the final Dyson Shell.
- **Void Mapping (Null)**: 7 new nodes from The Event Horizon to the Root Directory.
- **Path Purity**: Resources are now strictly path-specific. The Neural Bridge has been decommissioned in favor of absolute identity focus.

### 4. The GTC Conflict
Final confrontation with Director Vance in a 3-stage interactive climax:
- **Firewall Gauntlet**: Breach core defenses.
- **The Cage**: Survival mode during core isolation with high integrity drain.
- **Dead Man's Switch**: Philosophical showdown determining the final state of the world.

## 🎨 Sensory Engine
- **Industrial HUD**: Segmented Rack Gauges (LED block style) and Circular Bloom link matrices.
- **Kinetic HUD**: 120Hz optimized UI with hardware-isolated graphics layers.
- **Audio Engine 2.1**: Mobile-optimized Triangle wave sounds and path-specific ambient tracks.
- **CRT Aesthetics**: Scanline noise, vignettes, and identity-aware glitch effects.

## 🛠️ Technical Stack
- **Platform**: Android (API 26+)
- **Architecture**: Zero-Recomposition (Recomposition-isolated volatile stats).
- **Language**: Kotlin / Jetpack Compose
- **Persistence**: Room Database (v18 schema)
- **Serialization**: Kotlinx Serialization
